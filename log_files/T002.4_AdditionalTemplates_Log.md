## Key Features

### Google Fonts Integration
All templates use Google Fonts via @import for consistent typography:
```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&display=swap');
```

### Landscape Support
Three templates support landscape orientation:
```css
@page { size: landscape; margin: 1cm; }
```

### Dark Mode
Template 16 demonstrates dark mode with inverted colors:
```css
body { background: #111827; }
.invoice { background: #1f2937; color: #e5e7eb; }
```

### Borderless Design
Template 19 shows ultra-minimal borderless approach:
```css
/* No table borders, only subtle dividers */
.item-row { border-top: 1px solid #f4f4f5; }
```
