## Implementation Details

### Real Implementation
```csharp
public SdkResult<bool> InicializaSDK(string dataPath)
{
    if (_initialized)
    {
        return new SdkResult<bool>
        {
            Success = true,
            Value = true,
            ErrorMessage = "SDK already initialized"
        };
    }

    try
    {
        var result = MGW_InicializaSDK(dataPath);

        if (result == 0)
        {
            _initialized = true;
            return new SdkResult<bool> { Success = true, Value = true };
        }
        else
        {
            _lastError = $"SDK initialization failed with code {result}";
            return new SdkResult<bool>
            {
                Success = false,
                ErrorCode = result,
                ErrorMessage = _lastError
            };
        }
    }
    catch (Exception ex)
    {
        _lastError = ex.Message;
        return new SdkResult<bool>
        {
            Success = false,
            ErrorCode = -1,
            ErrorMessage = ex.Message
        };
    }
}
```

### Mock Implementation
```csharp
public SdkResult<bool> InicializaSDK(string dataPath)
{
    if (string.IsNullOrEmpty(dataPath))
    {
        return new SdkResult<bool>
        {
            Success = false,
            ErrorCode = -1,
            ErrorMessage = "Data path cannot be empty"
        };
    }

    _initialized = true;
    return new SdkResult<bool> { Success = true, Value = true };
}
```

### Error Codes
| Code | Meaning |
|------|---------|
| 0 | Success |
| -1 | Invalid path |
| -2 | SDK not found |
| -3 | License error |

## Notes
- Path typically: `C:\Contpaqi\Data`
- Must be called before any other SDK function
- Idempotent (safe to call multiple times)
