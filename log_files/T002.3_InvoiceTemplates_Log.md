## Jinja2 Placeholders Used

All templates include these placeholders:

```jinja2
{{ emisor.name }}       - Issuer company name
{{ emisor.rfc }}        - Issuer RFC (tax ID)
{{ emisor.address }}    - Issuer address
{{ receptor.name }}     - Recipient company name
{{ receptor.rfc }}      - Recipient RFC
{{ receptor.address }}  - Recipient address
{{ folio }}             - Invoice number
{{ date }}              - Invoice date
{{ currency }}          - Currency code (MXN)
{% for item in items %} - Line items loop
{{ item.description }}  - Item description
{{ item.quantity }}     - Item quantity
{{ item.unit_price }}   - Item unit price
{{ item.amount }}       - Item total amount
{% endfor %}
{{ subtotal }}          - Pre-tax total
{{ iva }}               - IVA tax amount (16%)
{{ total }}             - Final total
```
