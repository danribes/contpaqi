## Typography Variety

### Google Fonts Integration

```html
<head>
    <style>
        @import url('https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&display=swap');
        body { font-family: 'Roboto', sans-serif; }
    </style>

```

### Font Categories Used

| Type | Fonts | Use Case |
|------|-------|----------|
| Sans-serif | Roboto, Open Sans, Lato, Montserrat | Modern, clean |
| Humanist | Nunito, Poppins | Friendly, approachable |

## Page Orientation

### Portrait (Default)
```css
/* No special CSS needed - portrait is default */
body { padding: 30px; }
.invoice { max-width: 750px; }
```

### Landscape
```css
@page {
    size: landscape;
    margin: 1cm;
}

body { padding: 15px; }
.invoice { max-width: 1100px; }
```

### When to Use Landscape
- Many columns in items table
- Wide data (long descriptions)
- Dashboard-style layouts
- Corporate reports

## WeasyPrint Considerations

### Supported Features
- CSS Grid and Flexbox
- Google Fonts (@import)
- @page rules
- Border-radius
- Box shadows (limited)
- Gradients

### Not Supported
- JavaScript
- CSS animations
- External images (need embedding)
- Some advanced CSS filters

### Font Fallbacks
```css
font-family: 'Roboto', 'Arial', sans-serif;
font-family: 'Source Sans Pro', 'Georgia', serif;
```

## Testing Template Variety

### Check Unique Content
```python
def test_templates_are_different(self):
    templates = get_all_templates()
    for i, t1 in enumerate(templates):
        for j, t2 in enumerate(templates):
            if i < j:
                assert t1 != t2
```

## Best Practices

### 1. Font Loading
```css
/* Use display=swap for faster rendering */
@import url('...&display=swap');
```

### 2. Color Consistency
```css
/* Define colors once */
:root {
    --primary: #2563eb;
    --secondary: #64748b;
}
.header { background: var(--primary); }
```

### 3. Spacing System
```css
/* Use consistent spacing */
.section { margin-bottom: 25px; }
.subsection { margin-bottom: 15px; }
```

### 4. Typography Scale
```css
h1 { font-size: 28px; }
h2 { font-size: 20px; }
body { font-size: 11px; }
.small { font-size: 10px; }
```
